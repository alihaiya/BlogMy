## idea 

### 快捷键

``` 
Ctrl + F12 清单
ctrl + I  引入包或方法
Ctrl +Alt +T 提示生成方法 
Alt + Insert 类似eclipse 的source
Shift + F6 同名的全选修改  
Ctrl +e  切换文件
Ctrl +shift + 空格 代码补全 
Ctrl+H 查看类的继承关系，例如HashMap的父类是AbstractMap，子类则有一大堆
Ctrl+Alt+B查看子类方法实现
Alt+F7  查找类或方法在哪被使用
Ctrl+F/  其中Ctrl+F是在本页查找，
Ctrl+Shift+F   是全局查找
上一行Ctrl+Alt+enter
下一行shift+enter
ctrl + d 复制到下几行
ctrl + y 删除 
ctr+shift+u  大小写转化 
快捷键ALT + 7  打开左侧的Structure查看当前文件中的所有方法.
ctrl + shift + F12 放大编辑窗口
Ctrl + shift + n  类似于eclipse的 ctrl+shift+r
ctrl + n 类似于eclipse 的 ctrl + shitf + t
Ctrl + alt + left 方法跳转后再跳转回来
Ctrl+Shift+上下键   上下移动一行代码
alt+enter 导入包或主动抛出异常
ctrl + alt + o 删除无用的import包
ctrl + alt + L  代码格式化 
shift + F9  实时编译并启动tomcat
Alt + capslk + 鼠标左键   列选择

psvm 生成main主方法
sout 生成system.out.println();
Ctrl + Alt + T 提供的是代码块包裹功能 - Surround With。可以快速将选中的代码块，包裹到选择的语句块中。
iter 快捷生成增强型循环
itar
fori

```

### debug模式
```
IDEA 的 debug模式 
F9：执行
Alt+F10：显示执行断点
F8： 跳到下一步
F7：进入到代码
Alt+shift+F7：强制进入代码
Shift+F8：跳到下一个断点
Atl+F9：运行到光标处
ctrl+shift+F9：debug运行java类
ctrl+shift+F10：正常运行java类

Alt+F8：debug 选中代码计算某个表达式的值 
```
